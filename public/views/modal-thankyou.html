Este foi um demo, que surgiu de uma vontade própria de alguns nerds de mostrar um sistema de matrículas coerente com uma universidade do 'Séc XXI'.

Nossa vontade era de criar algo 100% funcional, mas percebemos que por questões burocráticas seria bem difícil esse projeto ser posto em prática. De qualquer forma, investimos 4 semanas para desenvolver essa interface e todo Backend (a parte do 'servidor'). Muita coisa precisa ser finalizada, e acreditamos que este projeto sendo 'open-source' ajudaria, e muito, a comunidade como um todo.

Curioso para ver o código fonte? Acesse aqui: github.com/UFABC-NoBox/NoKicks

Acreditamos que um sistema de matricula, não resolve o problema das vagas, mas deve sim, reduzir os chutes. É por conta disso, que implementamos a possibilidade de 'simular' sua posição em tempo real na matéria, a fim de evitar ser chutado.
